# mewgentic-tw

`mewgentic` 繁體中文在地化專案。

本專案的目標是將遊戲文字 CSV 的 `en` 欄位改為繁體中文，供遊戲內選擇英文語系時顯示中文內容。

## 使用方法
將release下載之後 將壓縮檔解壓縮後把data資料夾放到遊戲根目錄
目錄大概會長以下這樣
### data
### Mewgentics.exe
### resources.gpak
### steam_api64.dll

## 注意事項
因為我是將中文欄位改在英文欄位中，所以進遊戲前 右鍵遊戲->內容->一般->語言確認是英文
然後進遊戲後不要去改遊戲內語言，如果出問題了把data刪除，接著驗證遊戲完整性，或是直接全部刪除重新下載再來一次
